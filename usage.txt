___________________________________
All the shortcuts to work in emacs
___________________________________

1) ALT + B  -> switch the buffer in the current window
2) ALT + SHIFT + B -> Switch to buffer in the other window


3) ALT + F -> Load file in the current window
4) ALT + SHIFT + F -> Load file in the other window


5) ALT + W -> Switch to the other window

6) ALT + C -> Toggle between H file and CPP file in current window
7) ALT + SHIFT + C -> Toggle between H file and CPP file in other window

8) ALT + K -> Kills a buffer(ask's to save if modified)

9) CTRL + SPACE -> Set mark
10) CTRL + TAB -> Reindent block
11) CTRL + Q -> Copies the block
12) CTRL + F -> Pasts the block
13) CTRL + W -> Cuts th block

14) ALT + u -> Undo
15) ALT + SHIFT + ; -> Pops back to previouse location

16) ALT + J -> Jumps to fucntion by name
17) ALT + S -> Save the file
18) CTRL + S -> Start incremental search
       CTRL + S -> Moves to next occurence
       CTRL + R -> Moves to previouse occurence
19) ALT + O -> Query replace
20) ALT + L -> Replace inside block
21) ALT + M -> Looks for build.bat and executes it

22) ALT + [ -> Start recording a keyboard macro
23) ALT + ] -> End recording a keyboard macro
24) ALT + ' -> Replaying the macro

25) ALT + . -> Reflow a comment
26) ALT + N -> Goto the next error after a compile
27) ALT + SHIFT + N -> Goto previouse error after compile
28) ALT + G -> Goto line number
